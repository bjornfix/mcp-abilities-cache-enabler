MCP abilities for Cache Enabler operations.

== Description ==

Adds MCP/Abilities API operations for Cache Enabler status, settings, diagnostics, and cache clearing.

== Installation ==

Upload and activate the plugin on a WordPress site where the Abilities API exposure plugin is active.
